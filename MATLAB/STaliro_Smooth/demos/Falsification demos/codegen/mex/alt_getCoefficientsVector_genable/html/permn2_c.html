  187     /*  PERMN - permutations with repetition */
  188     /*    Using two input variables V and N, M = PERMN(V,N) returns all */
  189     /*    permutations of N elements taken from the vector V, with repetitions. */
  190     /*    V can be any type of array (numbers, cells etc.) and M will be of the */
  191     /*    same type as V.  If V is empty or N is 0, M will be empty.  M has the */
  192     /*    size numel(V).^N-by-N.  */
  193     /*  */
  194     /*    When only a subset of these permutations is needed, you can call PERMN */
  195     /*    with 3 input variables: M = PERMN(V,N,K) returns only the K-ths */
  196     /*    permutations.  The output is the same as M = PERMN(V,N) ; M = M(K,:), */
  197     /*    but it avoids memory issues that may occur when there are too many */
  198     /*    combinations.  This is particulary useful when you only need a few */
  199     /*    permutations at a given time. If V or K is empty, or N is zero, M will */
  200     /*    be empty. M has the size numel(K)-by-N.  */
  201     /*  */
  202     /*    [M, I] = PERMN(...) also returns an index matrix I so that M = V(I). */
  203     /*  */
  204     /*    Examples: */
  205     /*      M = permn([1 2 3],2) % returns the 9-by-2 matrix: */
  206     /*               1     1 */
  207     /*               1     2 */
  208     /*               1     3 */
  209     /*               2     1 */
  210     /*               2     2 */
  211     /*               2     3 */
  212     /*               3     1 */
  213     /*               3     2 */
  214     /*               3     3 */
  215     /*  */
  216     /*      M = permn([99 7],4) % returns the 16-by-4 matrix: */
  217     /*               99     99    99    99 */
  218     /*               99     99    99     7 */
  219     /*               99     99     7    99 */
  220     /*               99     99     7     7 */
  221     /*               ... */
  222     /*                7      7     7    99 */
  223     /*                7      7     7     7 */
  224     /*  */
  225     /*      M = permn({'hello!' 1:3},2) % returns the 4-by-2 cell array */
